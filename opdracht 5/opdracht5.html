Projecten

1. Website met een Game en Uitleg

Projectnaam: Game Web App

Beschrijving: Voor dit project heb ik een interactieve website ontwikkeld die een eenvoudige game bevat. De website biedt ook een uitgebreide uitleg over hoe de game werkt, inclusief de spelregels en strategieën om te winnen.

Kenmerken:

  • Game: Een browsergebaseerde game ontwikkeld met HTML5, CSS3 en JavaScript.
  • Uitleg: Gedetailleerde uitlegsectie met tekst en afbeeldingen om de spelregels te verduidelijken.
  • Responsive Design: De website is ontworpen om goed te werken op zowel desktop als mobiele apparaten.

2. CRUD Applicatie

Projectnaam: CRUD To-Do List

Beschrijving: Dit project demonstreert mijn beheersing van CRUD (Create, Read, Update, Delete) operaties. Ik heb een To-Do list applicatie ontwikkeld waarin gebruikers taken kunnen toevoegen, bekijken, bijwerken en verwijderen.

Kenmerken:

  • Frontend: Ontwikkeld met HTML, CSS, en JavaScript voor een intuïtieve gebruikersinterface.
  • Backend: Node.js en Express.js om de CRUD-functionaliteit te ondersteunen.
  • Database: MongoDB voor het opslaan van de taken.
  • API: RESTful API endpoints voor het uitvoeren van CRUD-operaties.
